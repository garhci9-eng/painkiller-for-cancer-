## [1.0.0] — 2025-02-22

### Added / 추가
- Initial release / 최초 릴리스
- Real-time ChEMBL compound search for 5 non-opioid pain targets  
  비마약성 암성 통증 타겟 5종 ChEMBL 실시간 검색
- 2D molecular structure visualization via RDKit  
  RDKit 기반 분자 구조 2D 이미지 표시
- Lipinski Rule of Five automatic filtering  
  Lipinski Rule of Five 자동 필터링
- AI-based side-effect risk scoring  
  AI 부작용 위험 점수 계산
- CSV export of all results  
  전체 결과 CSV 다운로드
- Local SQLite database for saving compounds  
  화합물 저장용 SQLite 로컬 DB
- Favorites and search history  
  즐겨찾기 및 검색 기록
- Direct SMILES input analysis  
  SMILES 직접 입력 분석
- Streamlit web interface with 5 tabs  
  5개 탭 구성 Streamlit 웹 인터페이스
- Docker + docker-compose support  
  Docker 원클릭 실행 지원
- GitHub Actions CI pipeline  
  GitHub Actions CI 파이프라인
