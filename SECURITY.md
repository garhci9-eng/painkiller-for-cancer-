## Reporting a Vulnerability / 취약점 신고

**[EN]**  
If you discover a security vulnerability, please do **not** open a public GitHub issue.  
Instead, report it privately via GitHub's [Security Advisory](https://github.com/YOUR_USERNAME/cancer-pain-drug-discovery/security/advisories/new) feature.

**[KO]**  
보안 취약점을 발견하셨다면 공개 GitHub 이슈로 등록하지 마시고,  
GitHub [Security Advisory](https://github.com/YOUR_USERNAME/cancer-pain-drug-discovery/security/advisories/new) 기능을 통해 비공개로 신고해 주세요.

---

## Scope / 범위

**[EN]** This is a research tool that processes only public ChEMBL data and locally stored compound information. It does not handle personal health data or patient records.

**[KO]** 이 프로그램은 공개 ChEMBL 데이터와 로컬에 저장된 화합물 정보만 처리합니다. 개인 건강 데이터나 환자 기록은 다루지 않습니다.

---

## Disclaimer / 면책

**[EN]** This software is for research purposes only and must not be used for clinical decision-making without expert review.

**[KO]** 이 소프트웨어는 연구 목적으로만 사용되며, 전문가 검토 없이 임상적 의사결정에 사용해서는 안 됩니다.
